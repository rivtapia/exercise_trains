# Technologies:

- Maven 3
- Spring boot 2.1.6
- Java 8
- Postman

# Run:

- Inside project folder, using maven plugin, execute in terminal:

  mvn spring-boot:run
  

# Testing

- With server running, open 'Postman' and import this file:

  /src/main/resources/postman/trains_exercise.postman_collection.json
